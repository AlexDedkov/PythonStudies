
Требования:
SQL Server. Database farmersmarkets запущенная на локальной машине

Инструкция по запуску и использованию приложения:
1. Запустить exe файл. Все требуемые зависимости установятся сами
2. Ввести данные пользователя в базе данных SQL SERVER.
3. Нажать кнопку Connect. При успешном подключении появится сообщение об успешном подключении, в иных случаях покажет ошибку.
4. После успешного подключения в маленьком окне будет выведен список таблиц в базе данных
5. Выбрать требуюмую таблицу.
6. Кнопки:
6.1 View Table. Просмотр данных таблицы. Размер окна можно менять в зависимости от размера таблицы.
6.2 Add Data. Кнопка работает только после того как вы выбрали и просматриваете таблицу. При нажатии на нее откроется небольшое окно где можно внести данные новой записи.
После этого нажмите Submit и если данные введын согласно требованиям SQL таблицы то они будут добавлены в таблицу.
6.3 Update Data. Кнопка работает только после того как вы выбрали и просматриваете таблицу. Выберете строчку которую хотите редактировать и нажмите на нее левой кнопкой мыши. 
Нажмите Update Data и в открывшемся окне отредактируйте данные таблицы. После этого нажмите Submit и если данные введын согласно требованиям SQL таблицы то они будут добавлены в таблицу.
6.4 Delete Data. Кнопка работает только после того как вы выбрали и просматриваете таблицу. Выберете строчку которую хотите удалить и нажмите на нее левой кнопкой мыши. 
Нажмите Delete Data и выбранная запись будет удалена.
6.5 Search Data. Кнопка работает только после того как вы выбрали и просматриваете таблицу. Нажмите на кнопку Search Data и в открывшемся окне укажите интересующие вас значения в таблице.
Можно указать максимальные и минимальные значения, любое количество критериев по столбцам. Если ничего не выбрать то выведутся все данные таблицы.
